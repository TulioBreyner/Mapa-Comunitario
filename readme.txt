MAPA COMUNITÁRIO - Turma 2023/1 de Ciência da Computação, IFTM Ituiutaba.
Projeto em grupo: Guilherme (Documentação do projeto), Gustavo (Auxilio geral) e Marco Túlio (Desenvolvimento e codificação).

Projeto desenvolvido é uma aplicação web, atualmente as tecnologias usadas são HTML, CSS, JavaScript, biblioteca js Leaflet, ChatGPT para auxiliar o desenvolvimento, Github pages para hospedar o projeto online (https://tuliobreyner.github.io/) e a IDE utilizada é o VS Code. 

O projeto conta com as seguinte funcionalidades prontas:
1- Pode-se adicionar e remover marcadores para N lugares (apenas editando o arquivo.json, assim limitando a capcidade do usario de sobrecarregar informações).
2- É possivel adicionar informações básicas, foto e link de rota para cada marcador.
3- É possivel filtrar locais a partir de seu tipo de contribuição para a sociedade.
4- Busca de locais por nome.
5- Arquivo .json para salvar os marcadoes e suas respectivas informações.

Não foi possivel implementar as seguintes funcionalidades:

6- Sistema de login para que apenas o usuario administrador consigua editar as informações dos locais.
7- Cor do marcador ser alterada mediante interação do usuario.
8- As fotos dos locais não são salvas junto das outras informações no arquivo.json, então essa funcionalidade só roda localmente.
